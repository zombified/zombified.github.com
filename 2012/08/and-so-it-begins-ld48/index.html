And so it begins LD48

August 24, 2012 Now I have to make a game with the theme 'Evolution' in 48 hours

Source can be found at github.com/zombified/ld48-24/

[20:00 August 24th]

Evolution.

Let's see what I can do!

[23:38 August 24th]

Well, after a few hours of racking my brain, possibly a short nap, and some down time with the missus, I have an idea. Not a very com­pli­cat­ed or original one, but one that I think has some potential.

Here's a first look at it:

Not much to see, and not much to do in it, but it's a start!

Time for bed.

[6:45 August 25th]

Rested, and ready to go!

My goal for today is to have the entirety of gameplay done:

  • Eat small blobs, grow bigger
  • Get hit by bigger blobs, grow smaller
  • Start overlay
  • Dead/End overlay
  • Pause overlay
  • Sounds

My goal for tomorrow:

  • Fix up the UI
  • Add some animations
  • Fancy shit up
  • Add bonus and penalty blobs

I'm certain that my plans will change, but here goes nothing!

[7:30 August 25th]

Refactored some of my code a little, and added in some basic collision detection.

Now it's time for a shower and breakfast. Let's start this day off right!

[11:46 August 25th]

The game mechanics are shaping up nicely! I think there's a little more tweaking I want to do before I start on the menu's and sounds, but it's (IMO) quite enjoyable for a quick 5 minutes of play.

[14:04 August 25th]

I've got the start and end screens func­tion­ing the way I want them, and I've pretty much stopped tweaking the mechanic. I think it is good enough the more I play it. There's lots of things that I could add in to it... like a blob that would keep you the same size, an ability to push or pull all the blobs aound the mouse and such.

I've also come to think that the game is an analogy for life, so I'm rolling with it.

I do need to come up with a better name than 'FLOW' though, as it's just my working title.

Here's the death screen:

[18:13 August 25th]

Quick update before I eat dinner. I was going kinda slow this afternoon, but the "high scores" are displaying and the player can ether their name.

I'm going to work on a easter egg next... :)

[21:58 August 25th]

Added the easter egg! Not going to say what it is or post a screenshot of it, of course!

Also added music. I am by no means a music guy, so it shouldn't surprise you that it's nothing fancy. Might add a little more tomorrow before I submit the game, but, honestly, I don't know.

Since I spent most of my time on sound stuff, don't really have a screen shot!

Next, I have a little intro text to put on the start screen, then we have to rename the project to something official like.

[0:28 August 26th]

Still going strong, decided to create a custom fixed-width font for more alignment!

[9:33 August 26th]

Woke up refreshed, had some eggs, and started digging into more polish!

I added a death sound, made the text a little more readable, added an in­struc­tion on how to quit the game (which is probably error prone -- not really sure how to detect OSs/Window Managers well, so the arrow and message may be on the wrong side of the screen for releases like Kubuntu), and changed the title! It is now called "And Thus She Spake".

Here's a screen shot, you can see some of the updated text and the new title:

[12:53 August 26th]

Well, with around 7 hours to go I've finished bundling and releasing to the extent that I'm going to do.

Here's a link to the entry (just the preview, not the voting page): And Thus She Spake

Here's my releases:

  • LÖVE File (Make sure to install [LÖVE] separatly)
  • OS/X
  • Win 32-bit
  • Win 64-bit

Linux users will have to install LÖVE separately and run the LÖVE File as I am not familiar with bundling on those platforms, and there aren't any straight forward directions to follow to bundle for LÖVE.
