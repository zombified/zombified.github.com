BD2 p4

What I've Done

  • Wow, really spent a lot of time studying collision detection (I discovered my previous utilities just weren't going to cut it). Found out that I'm a bit rusty on some math concepts, so I did a little review as well.
  • Finished implementing the Separating Axis Theorem for Circle to AABB shapes (and also AABB to AABB) -- N Tutorial A by Metanet Software Inc was probably THE most helpful resource on this subject.
  • Also decided to implement the library and a small test program in python + pygame. Why? I have less experience with Lua, and I'm finding that I can work faster in Python because of it (duh) -- basically I've just decided to use tools I know rather than experiment. I LOVE love2d, but I don't think a sweet api is enough for me, right now. Tech switches are dangerous when just trying to get something done, but I feel like I prototyped in love2d/lua, and now I'm actually building the game.
