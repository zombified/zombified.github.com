A New Host!

I decided to get off of Linode, and the first step is moving my website to a host of some sort. I decided that I didn't need all the features of a CMS or big framework (like drupal, plone, etc, etc), especially with the amount of embeddable web services out there!

Instead, I'm opting to statically generate all my sites (more secure, smaller, and less complex). To do this, I use a really nice application by the name of blogofile. This allows me to template a site, then add content with a somewhat standard formatting language (like markdown).

Another upside to generating a static website is that I can host it pratically anywhere for cheap. The host I ended up picking was Google AppEngine. I picked the AppEngine because it allows me to host a static site (with some fiddling) and allows me to make certain bits of my site dynamic as I need to. It also allows me to host my stuff for free (up to certain quotas, which my site will likely never exceed), and provides a fast distributed system that I don't have to manage at all.

Maybe in a future post I'll go into more detail about my setup, but for now I'm not done yet! I'm just getting started!

Python + Twisted + IRC

This was my first experience with the twisted library, and I have to say that it turned out pretty nice for creating an IRC bot.

For the most part, it's follows the design presented in the Writing Clients tutorial found on twistedmatrix.com.

The only thing of note is how I was able to implement events not explicitly defined in the twisted.words.protocols.irc.IRCClient class. It took me some digging (probably because I don't really understand Twisted very well), but I found that if the IRCClient sees an unknown IRC command, it will call a function in the form of irc_COMMAND

So, for instance, I needed to get the name list of everyone on a channel. I ended up writing the following code:

def irc_RPL_NAMREPLY(self, *nargs):
    print ""

def irc_RPL_ENDOFNAMES(self, *nargs):
    pass
    #print "END OF NAMES"

Now my bot will handle the RPL_NAMREPLY and RPL_ENDOFNAMES commands.

Python I <3 You

It's amazing this thing called Python.

I integrated identi.ca into this site in about 10 minutes over my lunch break. Amazing.

I used the python-twitter module, and it took a whole of 3 lines!

import twitter
api = twitter.Api(base_url='http://identi.ca/api')
statuses = api.GetUserTimeline('jkyl')

Which then produces a list of objects that contain a 'text' property that I can use to display the actual statuses.
