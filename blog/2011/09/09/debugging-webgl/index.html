Debugging WebGL

In experimenting with webgl, I've found the webgl-debug.js script (from the khronos group) extremely helpful.

You can find it, and directions for using it, at: www.khronos.org/webgl/wiki/Debugging

You can also find a link to the WebGL Inspector there as well, which seems like it might be a really useful tool, but with my small experiments thus far, I haven't had the chance to make use of it.

I also found this nice little utility from mrdoob: stats.js -- it provides FPS and Milliseconds-Per-Frame numbers (along with little charts) and can provide a memory size statistic too, but for that you need to pass special parameters into chrome/safari (at least at the time of this writing).
