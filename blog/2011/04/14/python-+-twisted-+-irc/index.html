Python + Twisted + IRC

This was my first experience with the twisted library, and I have to say that it turned out pretty nice for creating an IRC bot.

For the most part, it's follows the design presented in the Writing Clients tutorial found on twistedmatrix.com.

The only thing of note is how I was able to implement events not explicitly defined in the twisted.words.protocols.irc.IRCClient class. It took me some digging (probably because I don't really understand Twisted very well), but I found that if the IRCClient sees an unknown IRC command, it will call a function in the form of irc_COMMAND

So, for instance, I needed to get the name list of everyone on a channel. I ended up writing the following code:

def irc_RPL_NAMREPLY(self, *nargs):
    print ""

def irc_RPL_ENDOFNAMES(self, *nargs):
    pass
    #print "END OF NAMES"

Now my bot will handle the RPL_NAMREPLY and RPL_ENDOFNAMES commands.
